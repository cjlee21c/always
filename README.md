# 1. 과제 구현 설명
a. AI 챗봇 및 프롬프트 엔지니어링

ChatGPT API를 사용하여 챗봇을 만들었고 reviews.json이라는 크롤링 데이터와 따로 만든 더미데이터를 프롬프트에 주입하는 방식으로 하였습니다. 이때 이용자들에게 보다 객관적이고 실용성이 있는 답변을 제공시키기 위해서 가지고 있는 데이터만으로 답변, 수치를 이용한 답변 강조 등과 같은 추가적인 프롬프트 엔지니어링을 진행하였습니다.

b. 데이터 크롤링 및 정제

크롤링을 위해서는 Puppeteer Stealth를 사용하여 제가 참고하고 싶은 11번가 음식물 처리기 리뷰 페이지에 있는 데이터를 수집하였습니다. 이때 차단을 피하기 위해서 중간중간 pause와 느린 속도의 스크롤링을 사용하였습니다. 데이터를 추출할 때 너무 짧은 리뷰는 거르고 조건에 맞는 리뷰의 작성자 이름, 날짜, 내용 등을 가져왔습니다.

c. 백엔드 및 데이터 파이프라인

이런 데이터를 reviews.json이라는 파일에 넣고 ChatGPT를 이에 기반하여 작동시키기 위한 기본적인 Node.js 백엔드를 구현하였습니다. 데이터 파이프라인 같은 경우에는 크롤링을 통해 리뷰 데이터를 reviews.json에 로컬 저장하였고, 이를 server.js에서 백엔드 처리 후 App.jsx에서 시각화를 하게 하였습니다.

d. 개발 프로세스 및 AI 협업

Gemini: 해당 프로젝트의 전체적인 틀과 방향성은 Gemini를 통해 구성하였습니다. 제가 만들려고 하는 프로젝트의 세세한 디테일과 제가 원하는 포인트를 구체적인 outline 형태로 먼저 알려주었고, Gemini의 suggestion을 바탕으로 더 세부적인 디테일을 추가하였습니다. 이때 Claude Code에 참고시킬 md file도 함께 만들었습니다.

Claude Code: 그 이후 대부분의 개발 과정은 Claude Code와 함께 하였습니다. 제가 만들어놓은 md file을 토대로 Claude에게 plan mode를 사용하여 전체적인 structure와 plan을 다시 만들고 개발을 진행하였습니다.
